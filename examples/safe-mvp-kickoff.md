# Safe MVP Kickoff Example

This example shows the shape of a dry-run kickoff packet generated by:

```bash
python3 scripts/seed_factory_docs.py /path/to/example-app \
  --project-name "Example App" \
  --work-mode safe_mvp \
  --target-outcome "Ship the core onboarding flow with focused proof" \
  --dry-run \
  --print-kickoff
```

## Files

```text
DRY-RUN CREATE /path/to/example-app/AGENTS.md
DRY-RUN CREATE /path/to/example-app/docs/factory_config.md
DRY-RUN CREATE /path/to/example-app/docs/review_index.md
DRY-RUN CREATE /path/to/example-app/docs/codex_factory_protocol.md
DRY-RUN CREATE /path/to/example-app/docs/handoff_protocol.md
DRY-RUN CREATE /path/to/example-app/docs/build_ledger.md
```
