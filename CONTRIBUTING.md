# Contributing

Thank you for improving `software-factory`.

## Development Setup

This repository has no runtime package dependencies. Use Python 3.11 or newer
to run the bundled generator and tests.

```bash
python3 -m unittest discover -s tests
python3 -m py_compile scripts/seed_factory_docs.py
```

If you have the Codex skill-creator tooling installed, also run:

```bash
python3 /path/to/skill-creator/scripts/quick_validate.py .
```

That validator requires `PyYAML`.

## Pull Request Checklist

- Keep `SKILL.md` concise and focused on agent workflow.
- Move detailed policy matrices, examples, and templates into `references/`.
- Keep scripts deterministic, dependency-light, and covered by tests.
- Run the generator with `--dry-run` before changing generated-output behavior.
- Update `README.md`, `agents/openai.yaml`, and tests when public behavior changes.
- Do not commit generated bytecode, local virtualenvs, editor files, or seeded
  downstream factory docs from test runs.

## Skill Design Notes

This is a Codex skill, so the primary consumer is another agent. Prefer precise
instructions over broad explanation, and keep project-specific assumptions out
of the skill package.
